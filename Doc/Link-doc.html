Link Accessibility - How it Works

The Link Carbon Component is primarily a navigational element. The Enter key activates the link and navigates to the link destination. When a link is disabled the ARIA state is set to aria-disabled="true", which changes the appearance and the tabindex="-1" so it is ignored by assistive technologies. When the link becomes active the ARIA state changes to aria-disabled="false" and the link is present in the tab order.

Accessibility Considerations

  • Images that are used as a link should include the ARIA role="link", a tabinex="0" and alternative text.
  • The purpose of the link should be clear and easy to understand for all users.
