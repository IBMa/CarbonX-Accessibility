Breadcrumb Accessibility - How it Works

The Breadcrumb Carbon Component provides a way for the user to navigate Web content by providing a group of links to the parent Web pages that preceed the current page. The breadcrumb is contained within a navigation landmark region. The landmark region contains an ARIA label, aria-label="breadcrumb" to advise users of assistive technologies that they are navigating a breadcrumb. CSS is used for the visual separators between each link so they are not represented in the accessibility tree used by screen readers which prevents redunant and at times distracting verbosity.

Accessibility Considerations

  • If the current page element in the breadcrumb is not a link, add the ARIA attribute aria-current="page".
