Toggle Accessibility - How it Works

The Toggle Component is a checkbox that is designed as a toggle switch with a position of either on or off. The Tab key is used to move focus to each toggle, and a toggle component identified as disabled is ignored in the tab order. The Space key is used to turn the toggle on and off. When the toggle is in the on position, the ARIA state is aria-checked="true" and the label is set to "On". When the toggle is in the off position, the ARIA state is aria-checked="false" and the label is set to "Off".

Accessibility Considerations

  • Labels should be clear and concise.
  • Users should be warned if selecting a toggle component will cause a change in context.
