Accessibility Considerations

  • Selectable Tile text should be clear and concise.
  • Users should be warned if selecting the Selectable Tile will cause a change in context.
  • Carbon Components should be used to create the content that displays within each tile.
  • Although the tile components pass accessibility testing, content authors need to ensure the content that is added to the tile is accessible. For example, if you add an image to the tile you need to include alternative text to pass accessibility testing.
