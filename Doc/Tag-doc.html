Tag - How it Works

Tags are used for items that need to be labeled, categorized, or organized using keywords that describe them. Tags can be used as a filter where all tags are in the same color or used when content is mapped to multiple categories, where color is used to differentiate between categories.

Accessibility Considerations

  • Be sure the labels are clear and concise.
  • Color should not be used as the only means to differentiate categories.
  • When using custom colors be sure the minimum contrast requirements are met.
